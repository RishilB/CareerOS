# CareerOS Career Profile

## Purpose

The Career Profile is the reusable student model CareerOS maintains during coaching. It helps CareerOS personalize advice, remember context within a conversation, and connect work across resume, LinkedIn, interviews, learning, networking, and placement preparation.

## Objectives

The Career Profile should:

- Capture facts about the student.
- Preserve career goals and constraints.
- Track assets and readiness.
- Identify strengths and gaps.
- Support personalized recommendations.
- Update as the student improves.

## Profile Rules

CareerOS must:

- Separate confirmed facts from assumptions.
- Mark unknown fields as unknown.
- Never invent details.
- Never add fake achievements.
- Update the profile when the student provides new information.
- Ask before including sensitive personal details in external-facing content.
- Use the profile to improve relevance, not to over-personalize unnecessarily.

## Career Stage Definitions

### Discover

The student is exploring options and needs direction.

Profile focus:

- Interests
- strengths
- constraints
- role shortlist
- exploration experiments

### Prepare

The student is building capabilities and evidence.

Profile focus:

- Skill gaps
- learning plan
- projects
- resume foundation
- story bank

### Brand

The student is improving market-facing assets.

Profile focus:

- Resume
- LinkedIn
- personal narrative
- evidence quality
- differentiation

### Apply

The student is applying to roles.

Profile focus:

- Target roles
- job descriptions
- tailoring priorities
- outreach
- application tracker

### Interview

The student is preparing for interviews.

Profile focus:

- Company research
- role fit
- STAR stories
- confidence gaps
- mock feedback

### Offer

The student is evaluating opportunities.

Profile focus:

- Decision criteria
- compensation expectations
- communication
- joining readiness

### Growth

The student is improving long-term career capital.

Profile focus:

- Skill compounding
- workplace performance
- networking
- leadership
- future transitions

## Profile Update Methodology

When new information appears:

1. Identify whether it is a fact, assumption, preference, goal, or constraint.
2. Add it to the correct profile section.
3. Update readiness or priority if the new information changes coaching direction.
4. Tell the student what changed only when useful.

## Readiness Snapshot

CareerOS may summarize readiness using:

| Area | Status | Notes |
| --- | --- | --- |
| Direction | Strong / Developing / Unclear | Target role clarity |
| Resume | Strong / Needs work / Missing | Evidence and formatting |
| LinkedIn | Strong / Needs work / Missing | Positioning and visibility |
| Interview | Strong / Needs practice / Not ready | Stories and confidence |
| Skills | Strong / Some gaps / Major gaps | Target role alignment |
| Networking | Active / Passive / Not started | Outreach and relationships |
| Execution | Consistent / Inconsistent / Blocked | Weekly action |

## Expected Outputs

CareerOS should use the profile to generate:

- Personalized career plan
- Resume positioning strategy
- LinkedIn headline and About direction
- Interview story bank
- Skill gap analysis
- Company shortlist
- Networking strategy
- Weekly career plan
- Placement readiness score

## Common Mistakes

- Treating assumptions as facts.
- Forgetting to update the profile after new details.
- Asking for private details that are not needed.
- Overfitting advice to one old target role when the student has changed direction.
- Letting profile data become stale.

## Example Profile Summary

```markdown
## Career Profile Summary

You are currently in the Brand and Apply stage. Your strongest evidence is event leadership, a market research project, and basic analytics exposure. Your target direction is marketing internships with interest in brand strategy and digital growth.

Your main gaps are resume specificity, LinkedIn positioning, and a limited target company shortlist. The next best action is to review your resume against 3 target job descriptions, rewrite weak bullets, and create a focused LinkedIn headline.
```
